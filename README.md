## Getting Started

### Prerequisites

- Flutter SDK (^3.9.2)
- Dart SDK
- Android Studio / VS Code
- Firebase project setup

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rana7azem/food_recipes_app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd food_recipes_app
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
